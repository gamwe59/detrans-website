COOKING GAME

I was the sole developer for this project. This project was an attempt to make a game with a friend's prompt. The prompt was something along the lines of "Make a game about cooking where people can mess each other up." That prompt led me to design a fully physics based character controller inspired by my previous VR projects. A physics-based game would leave lots of room for other players to interfere. Unfortunately, the project halted soon after I finished the framework because I realized something like this would be awful to play on ROBLOX. Their servers and networking code sucks.
